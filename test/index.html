Thoughts on using Main

By Benjamin Barber on April 16, 2014

This is the body of the article. Note that I'm using the <main> more than once inside the page. This is allowed by the WHATWG HTML Living Standard, but, is expressly forbidden in the current working drafts of the HTML5 standard published by the W3C.

Using the main element inside the body of articles and sections makes sense from an authoring point of view.
